Dato interesante: herramientas con uv

uv permite ejecutar herramientas de Python, como por ejemplo ruff o pytest sin necesidad de instalarlas en el proyecto.

uvx — ejecutar sin instalar

Ejecuta una herramienta en un entorno temporal aislado y desaparece cuando termina.

uvx ruff check .
uvx pycowsay hola desde uv

uv tool install — instalar de forma permanente

si usas una herramienta constantemente, instálala globalmente para no tener que escribir uvx cada vez.

uv tool install ruff
ruff --version

actualizar herramientas

uv tool upgrade ruff
uv tool upgrade --all
